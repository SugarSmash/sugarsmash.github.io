# App Screenshots

To complete the website, save your app screenshots to this folder with these names:

## iPhone Screenshots (4 images in grid):
1. **graph-view.png** - The main glucose graph view (Image #1 or #11 from your screenshots)
2. **calendar-view.png** - Calendar view showing monthly data (Image #2 or #10)
3. **awards-view.png** - Awards page with achievement boxes (Image #4)
4. **settings-view.png** - Settings page with goal sliders (Image #3 or #9)

## iPad Screenshot (1 larger featured image):
5. **ipad-view.png** - iPad view showing the optimized layout (Image #5, #6, or #7)

## Recommended Screenshots:

- Use the **iPhone screenshots** (dark ones) for the grid - they look modern and professional
- Use an **iPad screenshot** for the featured image below to showcase the tablet experience
- Choose screenshots with **good data** showing the app in action
- Make sure screenshots are **high resolution** (actual device screenshots)

The website will automatically display these images:
- iPhone screenshots in a responsive 4-column grid
- iPad screenshot centered below as a featured larger image
